## :point_down:Steps to initialize the project:

- Clone the repository
```
$ git clone https://github.com/anjupriya-v/weather-app.git
```
- Redirect to the cloned repo directory

- Open up the terminal 

- Install the dependencies
```
npm install
```
- Start the project 
```
npm start
```
